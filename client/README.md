# TaskFlow Frontend (Client)

This is the React frontend for the TaskFlow application, built with [Vite](https://vitejs.dev/).

> **Note:** Please see the [Main Project README](../README.md) in the root directory for full documentation, setup instructions, architecture details, and API references.

## Development

To run the frontend development server:

```bash
npm install
npm run dev
```

The application will start at `http://localhost:5173` and expects the backend API to be running on `http://localhost:5000` (requests to `/api` are proxied).
